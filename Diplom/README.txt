- Основной задачей данного проекта является создание модели для предсказания стоимости домов по их истории.
- В датасете представлено 377185 домов с их параметрами.
- Для начала работы с датасетом надо очистить данные от ненужной информации и только после этого приступать к поиску новых параметров из уже сущестующих, если это требуется.После данного этапа наступает время создания модели, её обучение и тестирование на данных.
